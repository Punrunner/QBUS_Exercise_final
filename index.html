📘 Hypothesis Testing — Visual Guide

Step-by-step process, decision rules, and when to use each test.

The 5 Steps of Hypothesis Testing

1
State the hypotheses: Write H₀ (null) and H₁ (alternative). H₀ always has an equality (=, ≤, ≥). H₁ is what you want to prove.
2
Choose the test & significance level α: Is σ known → z-test. σ unknown → t-test. Typically α = 0.05.
3
Calculate the test statistic:
z-test: Z = (X̅ − μ₀) / (σ / √n)   t-test: T = (X̅ − μ₀) / (s / √n)
4
Make a decision: p-value approach: reject H₀ if p < α. Critical value approach: reject H₀ if |test stat| > critical value.
5
Conclusion in context: "At the 5% significance level, there is sufficient evidence that..." — always write in the language of the problem.

Type I vs Type II Errors

Decision \ RealityH₀ is TRUEH₀ is FALSE
Fail to reject H₀✅ Correct (confidence = 1−α)❌ Type II error (β)
Reject H₀❌ Type I error (α)✅ Correct — Power (1−β)
💡 Reducing α (stricter test) → fewer Type I errors but more Type II errors. Increasing sample size reduces BOTH types.

📐 L.I.N.E — Regression Diagnostics

The four assumptions of linear regression and how to check each one.

📏
L — Linearity
The relationship between X and Y must be linear (straight-line). Check: residual plot vs X should show NO pattern.
🔗
I — Independence
Residuals must be independent of each other. Check: Durbin-Watson statistic (should be ≈ 2). Critical in time-series data.
🔔
N — Normality
Residuals should be normally distributed. Check: Shapiro-Wilk test (p > 0.05) and Normal Q-Q plot (points on diagonal).
⚖️
E — Equal Variance
Variance of residuals must be constant (homoscedasticity). Check: residual plot should show uniform spread. Fan shape = violation.

What Each Residual Plot Pattern Means

Pattern in residual plotAssumption violatedFix
🌀 Curved / U-shapeLinearity (L)Add X² term, or log(X)
🌪️ Fan / funnel shapeEqual variance (E)log(Y), or weighted LS
📈 Systematic drift over timeIndependence (I)Include lagged variables
🏔️ Skewed or heavy tails on Q-QNormality (N)log(Y), check outliers
🌫️ Random scatter — no patternNone — all good!No fix needed
